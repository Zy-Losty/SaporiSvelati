# 🍽️ Sapori Svelati — Svelare l'Anima del Cibo

[![Sapori Svelati Logo](assets/images/brand/logo.svg)](https://zy-losty.github.io/SaporiSvelati/)

**Sapori Svelati** è un progetto di divulgazione gastronomica indipendente che esplora la storia, la cultura e le curiosità più affascinanti del mondo alimentare. Un viaggio narrativo che va oltre la semplice ricetta per scoprire le radici profonde delle nostre tradizioni culinarie.

---

## ✨ Caratteristiche Principali

- 📖 **Articoli Narrativi**: Approfondimenti su eccellenze come il *Wagyu*, storie di cocktail classici come il *Dry Martini* e la storia della *Via delle Spezie*.
- 🔍 **Catalogo Dinamico**: Un sistema di filtraggio intelligente che permette di navigare tra le categorie: Storia, Cultura, Alimenti e Curiosità.
- 📱 **Design Responsive**: Un'esperienza d'uso ottimizzata per ogni dispositivo (Mobile, Tablet, Desktop) con un approccio "Mobile-First".
- 🎨 **Estetica Premium**: Interfaccia moderna con micro-animazioni fluide, effetti di hover dinamici e un'impaginazione elegante curata nei minimi dettagli.
- 🍪 **Gestione Privacy**: Banner cookie integrato e pagine legali (Privacy & Cookie Policy) complete.
- 📧 **Newsletter**: Sistema di iscrizione integrato per ricevere storie e curiosità settimanali.

## 🛠️ Stack Tecnologico

Il progetto è realizzato con tecnologie web moderne ed efficienti:

- **HTML5**: Struttura semantica per SEO e accessibilità (Aria-labels).
- **CSS3 (Modular)**: Architettura CSS organizzata in moduli (`base`, `home`, `article`) per massima manutenibilità e performance.
- **Vanilla JavaScript**: Logica dinamica fluida senza dipendenze pesanti.
- **Font Awesome**: Iconografia elegante e moderna.
- **Google Fonts**: Tipografia scelta per la massima leggibilità (Inter/Roboto/Outfit).

## 📁 Struttura del Progetto

```text
SaporiSvelati/
├── assets/          # Logo, immagini articoli e icone
├── css/             # Moduli CSS (base.css, home.css, article.css)
├── js/              # Logica JavaScript (navigazione, filtri, animazioni)
├── pages/           # Singoli articoli del blog
├── index.html       # Landing page principale
└── catalogo.html    # Pagina catalogo con filtri
```

## 🚀 Come Visualizzare il Progetto

Puoi visualizzare il sito live direttamente su GitHub Pages oppure clonare la repository e aprire il file `index.html` nel tuo browser:

```bash
git clone https://github.com/Zy-Losty/SaporiSvelati.git
cd SaporiSvelati
# Apri index.html con il tuo browser preferito
```
